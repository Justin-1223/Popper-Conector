# Popper Connector Setup

This directory contains the software and instructions needed to connect the
Popper coffee roaster to Artisan through a Raspberry Pi.

## Directory layout

- `pi-side-code/` contains software that runs on the Raspberry Pi connected to
  the roaster hardware.
- `computer-side/` contains software and configuration that runs on the
  computer using Artisan.

Detailed installation, configuration, connection, and verification steps will
be added here as the software is developed.
