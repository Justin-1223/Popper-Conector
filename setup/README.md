# Popper Connector Setup

This directory contains the software and instructions needed to connect the
Popper coffee roaster to Artisan through a Raspberry Pi.

## Directory layout

- `pi-side-code/` contains the Raspberry Pi service, configuration, startup
  unit, and complete installation instructions.
- `computer-side/` contains software and configuration that runs on the
  computer using Artisan.

Start with the README in the directory for the side you are setting up. Set up
the Raspberry Pi first, then install Artisan and load the supplied computer-side
settings file.
