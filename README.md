# Popper-Conector
This project transforms the Popper coffee roaster into a computer-controlled roasting system by integrating a Raspberry Pi, thermocouple sensing, and Artisan roasting software. The system monitors bean temperature in real time, logs roast data, and electronically controls the roasters heat to enable repeatable, high-quality coffee roasting

Quick Start For Roasting
Use this section if you just want to roast coffee.

Plug in the Popper.
Plug in the Popper control box / Raspberry Pi.
On your computer, join the Wi-Fi network named PopperRoaster.
Use the Wi-Fi password roastcoffee.
Open Artisan.
Load the Popper profile.
Make sure the temperature is updating in Artisan.
Press ON.
Press START.
Add beans and press CHARGE.
Watch the roast curve.
At the end, press DROP.
When finished, press OFF.

Important notes:

Your computer may lose normal internet while connected to PopperRoaster.
That is normal.
Do not start roasting if temperature is not updating in Artisan.
Never leave the roaster unattended.

If Artisan does not connect:

Check that your computer is connected to the PopperRoaster Wi-Fi network.
Close and reopen Artisan.
Unplug and replug the Pi/control box.
Wait 30 seconds.
Reconnect to PopperRoaster.
Reopen Artisan.
If anything seems wrong during roasting:

Press OFF in Artisan if possible.
Let the Popper cool.
Unplug the Popper only if needed.
Do not reach into the hot roaster.
