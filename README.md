# Popper-Conector
This project transforms the Popper coffee roaster into a computer-controlled roasting system by integrating a Raspberry Pi, thermocouple sensing, and Artisan roasting software. The system monitors bean temperature in real time, logs roast data, and electronically controls the roasters heat to enable repeatable, high-quality coffee roasting
